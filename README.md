# skills 自建

这是一个本地自建 skills 仓库，用于存放可被 Skills CLI 安装的技能。

## 目录

- `storyboard-director/`: 分镜导演（剧本 -> 文字分镜表 -> 参考图 -> 分镜帧 -> 导出）

## 安装（示例）

如果你把本仓库推到 GitHub（例如 `https://github.com/<you>/skills-zijian`），可以用下面方式安装：

```bash
npx skills add https://github.com/<you>/skills-zijian --skill storyboard-director -g -y
```
