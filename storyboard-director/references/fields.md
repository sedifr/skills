# 文字分镜表字段清单（默认全字段）

用途：在每个项目创建/更新 `文字分镜表.xlsx` 之前，先把本清单展示给用户，让用户决定是否增删列。

说明：

- “必备列”用于导演调度、一致性、生成、排版的闭环。
- “可选列”用于更影视工业化的控制（焦段/光向/轴线等）。

## 必备列（推荐默认开启）

1. 镜头号（01/02/03...）
2. 场次ID（S01/S02...）
3. 内/外（内/外 或 INT/EXT）
4. 地点
5. 时间（日/夜/黄昏/清晨...）
6. 景别（远景/全景/中景/近景/特写 或 ELS/WS/MS/CU/ECU）
7. 机位/角度（平视/俯视/仰视/过肩/主观等）
8. 运镜（静/推/拉/摇/移/跟/手持/升降等）
9. 焦段（倾向或毫米，例：广角/标/长 或 24/35/50/85）
10. 构图要点（主体位置、视线、前中后景、负空间）
11. 人物（可逗号分隔）
12. 动作/调度（走位、关键动作、cut on action 点）
13. 台词/VO
14. 声音/氛围（环境音、音乐提示、音效点）
15. 节奏/时长（快/中/慢 或 1-2s/2-3s）
16. 转场/剪辑点（硬切/叠化/匹配剪辑/动作剪辑点）
17. 连续性备注（轴线、方位、光向、道具位置、服装状态）
18. 图片文件（回填相对路径）
19. 图片预览（嵌入缩略图）
20. 生成提示词（英文）
21. 使用参考图（自动填或手动覆盖：场景/人物引用）
22. 生成模型/参数（温度等）
23. 状态（未生成/已生成/需重做/已定稿）
24. 重做原因/备注

## 可选列（按项目需要增删）

- 摄影机高度/机位高度（low/eye/high 或 cm）
- 轴线方向（A-to-B）
- 视线方向（左/右/向上/向下）
- 镜头速度（slow/normal/fast）
- 光向（左前/右后/顶光/逆光）
- 色板/主色（3-5 色）
- 关键道具清单（用于一致性）
- VFX/特效备注（冻结/时间停止/魔法等）
- 合成/拍摄备注（实拍分区合成、道具固定方式等）
